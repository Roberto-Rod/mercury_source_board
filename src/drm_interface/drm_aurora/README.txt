The files in the sub-directories core/gt/support are generated by the Xilinx IP CORE Generator. 

If the core is re-configured then the files must be copied from the CORE Generator output directories to the src/drm_interface/drm_aurora directory and its sub-directories as follows:

Source                                                   | Destination
---------------------------------------------------------|--------------------
ipcore_dir/drm_aurora.vhd                                | drm_aurora
ipcore_dir/drm_aurora/src/*                              | drm_aurora/core
ipcore_dir/drm_aurora/example_design/gt/*                | drm_aurora/gt
ipcore_dir/drm_aurora/example_design/cc_manager/*        | drm_aurora/support
ipcore_dir/drm_aurora/example_design/clock_module/*      | drm_aurora/support
ipcore_dir/drm_aurora/example_design/traffic_gen_check/* | drm_aurora/support
